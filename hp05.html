1. Crear archivo hp05.

Sigue estos pasos para clonar un repositorio, crear una rama, hacer un cambio y abrir un pull request.

1. Clonar el repositorio

Copia la URL HTTPS del repositorio en GitHub y clónalo localmente:

git clone https://github.com/usuario/mi-repo.git

2. Crear y cambiar a una nueva rama

Crea una rama para trabajar en tu cambio (ejemplo: feature):

git checkout -b feature

3. Hacer cambios y confirmar (commit)

Haz los cambios en los archivos, luego añade y confirma los cambios:

git add .
git commit -m "Descripción del cambio"

4. Subir la rama al origen (GitHub)

git push -u origin feature

5. Crear un Pull Request

En GitHub, abre un Pull Request desde tu rama (feature) hacia la rama principal (main o develop).

6. Revisar y fusionar

Revisa los comentarios, realiza ajustes si hace falta y fusiona (merge) cuando esté aprobado.
